# 운영체제 기본 명령어의 개요
- 운영체제를 제어하는 방법은 크게 CLI와 GUI로 구분할 수 있다.
- CLI는 키보드로 명령어를 직접 입력하여 작업을 수애하는 사용자 인터페이스를 의미한다.
- GUI는 키보드로 명령어를 직접 입력하지 않고, 마우스로 아이콘이나 메뉴를 선택하여 작업을 수행하는 그래픽 사용자 인터페이스를 의미한다.
